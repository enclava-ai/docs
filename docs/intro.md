# Enclava 

Enclava, a platform dedicated to making privacy practical and using confidential inference as simple as possible.

## What is Enclava?

Enclava provides a simple way to run AI on sensitive data without exposing it. Process documents, conversations, and private information while keeping everything encrypted and secure.
